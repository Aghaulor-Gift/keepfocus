# KeepFocus

A responsive one-page marketing website for **KeepFocus**, a productivity application designed to help users organize tasks, manage priorities, track progress, and stay focused throughout the day.

## Project Description

KeepFocus is a modern landing page built with semantic HTML5 and CSS3. The website highlights the application's core features and encourages users to get started through a clear and engaging call-to-action.

The project demonstrates responsive web design principles using Flexbox, CSS Grid, media queries, and semantic HTML elements.

## Features

### Hero Section

* Clear and engaging headline

* Primary and secondary call-to-action buttons

### Features Section

* Smart Task Management
* Daily Focus Planner

* Smart Reminders
